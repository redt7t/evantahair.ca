Sea Raiders

The Objective

The objective of this project was to design and build an original game. There were three deliverables to the project; build a working, low-fidelity, prototype, design and build a finished version, then conduct user-testing and submit a final version that incorporates the feedback.

The Game

Sea Raiders is a game where you play as a pirate ship and you must defend your seas from incoming enemy ships. If you allow too many to get behind you, you will lose. Survive the onslaught long enough and you will face the enemy's largest ship, which you must defeat to win the game. The game can be played on desktop or mobile.

User Testing

We conducted user testing to determine what users thought about our game and what we could improve on. It included a quantitative questionnaire and qualitative feedback. The results of the questionnaire indicated that the difficulty of the game was well balanced, meaning we did not need to adjust it for the final iteration. The qualitative feedback indicated that users did not understand what the game controls were, so we incorporated this feedback and added the controls to the title screen.
